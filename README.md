# Online_Book_store

## Tools Used 
* PostgreSQL
* pgAdmin 4
* CSV Files

## Description
* This project is designed to manage book inventory, customer information, and orders efficiently using stucture SQL queries.
* The system allows storing detais such as book title ,genre , price , stock,customer, data, and order history.
* Various SQL operation like CREATE , INSERT, SELECT , JOIN,WHERE and GROUP BY, are used to retrieve meaningful information from the database.

## Features
* Create tables with primary & foreign keys
* Import CSV data
* SQL queries using JOIN, WHERE, GROUP BY
